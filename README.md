## ğŸ“¦ Installation

### Windows
1. Download `SocialVault-x.x.x-Windows-x64-Setup.exe`
2. Run the installer and follow the prompts
3. Launch Social Vault from your Start Menu

### macOS
1. Download the `.dmg` file for your Mac:
   - **Apple Silicon (M1/M2/M3):** `SocialVault-x.x.x-macOS-arm64.dmg`
   - **Intel:** `SocialVault-x.x.x-macOS-x64.dmg`
2. Open the DMG and drag Social Vault to Applications
3. Launch from Applications folder

### Linux
1. Download `.AppImage` or `.deb` for your distribution
2. For AppImage: Make it executable and run
   ```bash
   chmod +x SocialVault-x.x.x.AppImage
   ./SocialVault-x.x.x.AppImage
   ```
3. For .deb: Install with your package manager
   ```bash
   sudo dpkg -i SocialVault-x.x.x.deb
   ```
